# Banking System Project (Python)

## Overview
This is a simple banking system program written in Python. It simulates basic banking operations using a menu-driven interface.

## Features
- Deposit money into account
- Withdraw money with balance validation
- Check current balance
- Handles insufficient funds
- Uses functions and loops for program structure

## How it works
The program displays a menu to the user and allows them to choose a transaction. The balance is updated after each operation.

## Concepts Used
- Functions
- While loops
- Conditional statements (if/elif/else)
- User input handling
- Basic arithmetic operations
